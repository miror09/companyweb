# MirIS – Jednoduché weby pre každého

Moderný landing page pre slovenský webový projekt MirIS, ktorý ponúka jednoduché a profesionálne webové stránky pre malé podniky a jednotlivcov.

## O projekte

Toto je statický HTML landing page vytvorený pre prezentáciu služieb webového studia MirIS.

## Služby

- **Landing pages** – jedna prehľadná stránka
- **Firemné stránky** – prezentácia služieb a cien
- **Portfólio weby** – osobné stránky a projekty
- **Úpravy a údržba** – rýchle zmeny a aktualizácie

## Kľúčové vlastnosti

- Responzívny dizajn (mobile-first)
- Moderný tmavý dizajn s gradientmi
- Plynulé animácie a prechody
- Rýchle načítanie (žiadne frameworky)
- Interaktívne prvky (hamburger menu, theme toggle)
- PWA ready
- Kontaktný formulár
- Počítadlo návštev

## Technológie

- Čistý HTML5 a CSS3
- JavaScript (vanilla)
- Google Fonts (Inter)
- CSS custom properties
- Moderné CSS techniky (backdrop-filter, gradienty, animácie)
